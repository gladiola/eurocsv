# EuroCSV — Convertisseur CSV de localisation

EuroCSV aide les équipes à convertir des fichiers CSV entre différents formats internationaux. L'application adapte les délimiteurs, les séparateurs décimaux, les séparateurs de milliers et les formats de date afin que les données s'importent correctement dans la région cible.

> **Remarque:** fr-FR utilise généralement `;` comme délimiteur de champs, `,` comme séparateur décimal, `space` comme séparateur de milliers et `dd/MM/yyyy` comme format de date.
