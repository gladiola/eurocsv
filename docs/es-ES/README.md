# EuroCSV — Convertidor de localización CSV

EuroCSV ayuda a los equipos a convertir archivos CSV entre formatos internacionales de uso empresarial. Ajusta delimitadores, separadores decimales, separadores de miles y formatos de fecha para que los datos se importen correctamente en la configuración regional de destino.

> **Nota:** es-ES usa normalmente `;` como delimitador de campos, `,` como separador decimal, `.` como separador de miles y `dd/MM/yyyy` como formato de fecha.
