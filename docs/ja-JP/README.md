# EuroCSV — CSVローカライズコンバーター

EuroCSV は、国際的な業務フォーマット間で CSV ファイルを変換できるようにするツールです。区切り文字、小数点、千位区切り、日付形式を調整し、変換先のロケールで正しく取り込めるようにします。

> **注記:** ja-JP では通常、フィールド区切り文字に `,`、小数点に `.`、千位区切りに `,`、日付形式に `yyyy/MM/dd` を使用します。

## 使い方

CSVファイルをアップロードし、変換元と変換先のロケールを選択して、変換済みファイルをダウンロードしてください。
