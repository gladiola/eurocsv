# EuroCSV — Convertitore di localizzazione CSV

EuroCSV aiuta i team a convertire file CSV tra formati internazionali utilizzati nel lavoro quotidiano. L'applicazione adatta delimitatori, separatori decimali, separatori delle migliaia e formati data, così i dati vengono importati correttamente nella destinazione scelta.

> **Nota:** it-IT usa normalmente `;` come delimitatore di campo, `,` come separatore decimale, `.` come separatore delle migliaia e `dd/MM/yyyy` come formato data.
