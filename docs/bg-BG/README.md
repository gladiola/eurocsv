# EuroCSV — Конвертор за локализация на CSV

EuroCSV помага на екипите да преобразуват CSV файлове между международни бизнес формати. Приложението настройва разделителите на полета, десетичните знаци, разделителите за хиляди и форматите на датите, за да може данните да се импортират правилно в целевите регионални настройки.

> **Забележка:** bg-BG обикновено използва `;` като разделител на полета, `,` като десетичен разделител, `space` като разделител за хиляди и `dd.MM.yyyy` като формат на датата.
