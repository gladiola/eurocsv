# EuroCSV — CSV-lokaliseringskonverterare

EuroCSV hjälper team att konvertera CSV-filer mellan internationella affärsformat. Verktyget anpassar avgränsare, decimaltecken, tusentalsavgränsare och datumformat så att data importeras korrekt i målmiljön.

> **Obs:** sv-SE använder vanligtvis `;` som fältavgränsare, `,` som decimaltecken, `space` som tusentalsavgränsare och `yyyy-MM-dd` som datumformat.
