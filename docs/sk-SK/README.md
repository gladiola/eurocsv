# EuroCSV — Konvertor lokalizácie CSV

EuroCSV pomáha tímom prevádzať CSV súbory medzi medzinárodnými obchodnými formátmi. Upravuje oddeľovače, desatinné značky, oddeľovače tisícov a formáty dátumu, aby sa údaje správne importovali do cieľových miestnych nastavení.

> **Poznámka:** sk-SK zvyčajne používa `;` ako oddeľovač polí, `,` ako desatinný oddeľovač, `space` ako oddeľovač tisícov a `dd.MM.yyyy` ako formát dátumu.
