# EuroCSV — CSV-lokaliseringskonverter

EuroCSV hjælper teams med at konvertere CSV-filer mellem internationale forretningsformater. Programmet tilpasser afgrænsere, decimalseparatorer, tusindtalsseparatorer og datoformater, så data importeres korrekt i mållandestandarden.

> **Bemærk:** da-DK bruger normalt `;` som feltafgrænser, `,` som decimalseparator, `.` som tusindtalsseparator og `dd-MM-yyyy` som datoformat.
