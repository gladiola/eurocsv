# EuroCSV — Pretvarač lokalizacije CSV

EuroCSV pomaže timovima pretvarati CSV datoteke između međunarodnih poslovnih formata. Aplikacija prilagođava razdjelnike, decimalne znakove, odjelnike tisuća i formate datuma kako bi se podaci ispravno uvezli u ciljnu regionalnu postavku.

> **Napomena:** hr-HR obično koristi `;` kao razdjelnik polja, `,` kao decimalni znak, `.` kao odjelnik tisuća i `dd.MM.yyyy` kao format datuma.
